## üêü Proprietary Technology

### **.fsh Agent System** (DeepFish Innovation)
- **Dual-file architecture**: Factory settings + User training
- **Portable agents**: Agents as sellable, shareable files
- **Runtime merging**: No disk blending, clean separation
- **Factory reset**: Delete `.user.json` to restore defaults

### **Asset Bus Pattern** (Inter-Agent Communication)
- Agents pass generated assets via token IDs
- Clean, efficient data sharing
- No prop drilling or global state pollution

### **VSync TTS** (Visual Speech Synchronization)
- Phoneme-to-viseme mapping
- Real-time lip-sync animation
- Agent personality visualization
