### Prerequisites
- Node.js 18+
- npm or yarn
- API keys (see setup)

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/DeepFish.git
cd DeepFish

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
# Edit .env and add your API keys

# Run development server
npm run dev:all
```

This starts:
- **Frontend** on `http://localhost:3000`
- **API Proxy** on `http://localhost:3001`
